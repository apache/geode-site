Join Our Community of Contributors!

The Apache Geode team welcomes contributors who want to support the Geode technology. Our community builds everything from this website, from the Geode code to documentation and best practices information.

We especially welcome additions and corrections to the documentation, wiki, and website to improve the user experience. Bug reports and fixes and additions to the Apache Geode code are welcome. Helping users learn best practices also earns good karma in our community.

Mailing-lists

Users

Perfect if you build apps against Apache Geode or deploy Apache Geode, and a great place to start if you have a question.

To subscribe, send a blank email to
user-subscribe@geode.apache.org.

You can also read the archives.

Dev

If you are building contributions and modifications to Apache Geode, this is the list for you.

To subscribe, send a blank email to
dev-subscribe@geode.apache.org.

You can also read the archives.

Commits

This list receives an email whenever new code is contributed to Apache Geode.

To subscribe, send a blank email to
commits-subscribe@geode.apache.org.

You can also read the archives.

Issues

This list receives an email when an Apache Geode issue is created or updated.

To subscribe, send a blank email to
issues-subscribe@geode.apache.org.

You can also read the archives.

Events

Watch this space for upcoming Apache Geode events!

Want to organize an event? Subscribe to the Apache Geode Users list by sending a blank email to
user-subscribe@geode.apache.org, then email the list with your event idea!
